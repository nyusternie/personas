# Getting Started

## Quick Start

1. **Visit our web app** at [app.personas.example.com](https://app.personas.example.com)
2. **Create your persona** using passkey authentication
3. **Fund your wallet** with BCH or any supported tokens
4. **Start using AI commands** to manage your finances

## Technology Stack

Personas is built with modern web technologies:

- **Frontend**: Nuxt.js + Vite
- **Styling**: Tailwind CSS v4
- **Smart Contracts**: CashScript v0.11.5
- **Authentication**: WebAuthn/Passkeys
- **Blockchain**: Bitcoin Cash
