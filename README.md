## ğŸ’¡ Smart Account Architecture

Personas implement a unique Account Abstraction scheme specifically designed for the Bitcoin Cash and Nexa blockchains. This innovative approach separates transaction sponsorship from user interaction, enabling:

- **Gas Delegation**: Transactions can be sponsored by third parties
- **Batch Operations**: Multiple actions in single transactions
- **Session Keys**: Temporary permissions for dApp interactions
- **Social Recovery**: Custom authentication and recovery logic

Our architecture builds upon Bitcoin Cash's robust security while introducing Ethereum-like smart account capabilities, creating the perfect balance of security and usability.
